HTML 1일차

환경설정 (Eclipse, Tomcat 연동)

  1. 공유 폴더에 apache-tomcat-9.0.6 다운로드

  2. 내PC - 속성 - 고급 시스템 설정 - 환경변수에서 변수 설정




  3. Eclipse 에서 Window - Preferences 클릭




  4. Server - Add - Apach.Tomcat.9.0 선택 후 Finish




  5. Servers 탭 - 마우스 우클릭 - New - Server - Finish




  6. 등록한 서버 더블 클릭 - HTTP 포트 8080 에서 8081로 변경




    변경 이유 : Oracle에서 8080포트를 사용하기 때문에 추후 에러상황 회피 위함

프로젝트 및 html 생성

  1. New - Project - Web - Dynamic Web Project 클릭

  2. WebContent 폴더 밑에서 html file 생성

오늘 배운 html태그

태그명 내용 비고

doctype

 문서 형식

html

 웹 페이지의 시작과 끝

meta

 문자 인코딩 및 문서 키워드, 요약정보

title

 웹 페이지 제목

body

 브라우저에 실제 표시되는 내용

h1 ~ h6

 문서에서 제목을 정의 block 요소

div

 특정 영역, 그룹화 할 때 쓰임 = layout block 요소

p

 단락 block 요소

a

 하나의 페이지에서 다른 페이지를 연결, 하이퍼링크를 정의할 때 사용 p태그와 혼동X
inline 요소

br

 줄바꿈

ul

 순서가 없는 list 디폴트로 ● 붙음

ol

 순서가 있는 list 디폴트로 숫자 1부터 속성 값으로 알파벳, 원하는 숫자부터 가능

li

 ul 과 ol 안에서 각 항목을 나열할때 쓰임 block 요소

table

 셀들의 행과 열로 구성된 2차원 테이블을 정의

thead

 테이블에서 헤더 컨텐츠들을 하나의 그룹으로 묶을때 사용

tbody

 테이블에서 내용 컨텐츠들을 하나의 그룹을 묶을때 사용

tr

 테이블에서 셀들로 이루어진 하나의 행

th

 테이블에서 제목이 되는 헤더 셀 thead태그 내부에서 사용

td

 테이블에서 하나의 데이터 셀 tbody태그 내부에서 사용

input

 사용자로부터 입력을 받을 수 있는 입력 필드 form태그 내부에서 사용됨

form

 사용자로부터 입력을 받을 수 있는 HTML 입력 폼 HTML에서 유일하게 기능역할 하는 태그
